# Grocery List App

## Project Overview
This is a **Grocery List App**, built using **HTML, CSS, and JavaScript**, designed to help users manage their grocery items efficiently. Users can add, edit, and delete items from their list, as well as clear the entire list.  

The project demonstrates basic **DOM manipulation**, **event handling**, and **interactive UI development** in a web application. This project was built as a learning exercise to improve front-end development skills and understand dynamic JavaScript interactions.

---

## Features

- **Add Items:** Users can input a grocery item and add it to the list.
- **Edit Items:** Edit existing items directly in the list using the edit icon.
- **Delete Items:** Remove individual items using the delete icon.
- **Clear All Items:** Clear the entire list with a single button click.
- **Responsive Design:** Works on desktop, tablet, and mobile devices.
- **Interactive UI:** Uses hover effects and icons for a better user experience.
- **Iconography:** Integrated **Ionicons** for edit and delete actions.

---

## Technologies Used

- **HTML5** – Structure and layout of the application.
- **CSS3** – Styling, layout, responsive design, and hover effects.
- **JavaScript** – DOM manipulation, event handling, and interactive functionality.
- **Ionicons** – Vector icons for edit and delete actions.

## How to Use

Clone the repository to your local machine. Open `index.html` in your browser to launch the app.  

Type a grocery item in the input field and click the "Submit" button to add it to your list.  

To edit an existing item, click the pencil icon next to the item and enter the new value.  

To delete an item, click the trash bin icon next to it.  

To remove all items at once, click the "Clear items" button.
